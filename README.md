# Smart Attendance System Using Face Recognition 

An AI-powered attendance management system that automatically marks student attendance using Face Recognition. The system detects and recognizes registered students through a webcam and records attendance efficiently.

## Features

- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Student Registration
- Real-time Webcam Recognition
- Attendance Stored in Excel
- Simple User Interface
- Fast and Accurate Recognition

## Technologies Used

- Python
- Flask
- OpenCV
- face_recognition
- NumPy
- Pandas
- OpenPyXL

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/smart-attendance-system-face-recognition.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```
